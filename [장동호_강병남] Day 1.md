# Mac 팁

기본으로는 Window Ctrl -> Mac cmd 으로 치환
cmd + <— : 문장 제일 앞으로 가는 단축키
cmd + shift + 3 \ 4 \ 5 : 전체 \ 특정 \ 녹화 스크린샷

### 사파리
Cmd + n : 새로운 창
Cmd + t : 새로운 탭
Cmd + 1 \ 2 … : 1 \ 2… 번째 창

한 손가락으로 탭 클릭 설정
시스템 환경설정 > 트랙패드 > 포인트 및 클릭 > 탭 하여 클릭하기 토글
인터넷 계정 추가 설정
시스템 환경설정 > 인터넷 계정 > 계정 추가
미세한 기기설정
Option + shift + f1 \  f2 ….
